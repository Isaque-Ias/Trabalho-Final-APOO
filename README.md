## Objetivo

> Trazer uma forma interativa e estimulante que leve estudantes e futuros ingressantes do IFRN a aprender sobre os principais assuntos abordados no exame de seleção necessário para a qualificação do indivíduo a entrar na institução.

## Instruções de Navegação

> A pasta ^^^doc^^^ guarda a documentação e diagramação da estrutura do projeto.
> Nela, é possível encontrar o diagrama de casos de uso, diagrama de classe e o diagrama relacional, para estruturar o banco de dados.
> Além disso, essa mesma pasta contém arquivos documentativos, como a visão do produto e os requisitos funcionais e não-funcionais.
> Voltando para o root, a pasta ^^^src^^^ contém o produto, denominado ^^^app^^^, que segue uma estrutura de programação em camadas.
> O No ^^^app^^^, podemos encontrar:
>> Templates de páginas web.
>> Modelos de objetos importantes para a estrutura do site.
>> Mediador de interação pelo arquivo ^^^view.py^^^.
>> Por fim há o inicializador da aplicação ^^^index.py^^^.

# Como inicializar

>> abra o CMD no arquivo root da aplicação.
>> Instale os módulos no arquivo ^^^requirements.txt^^^ através do comando ^^^pip install -r src\app\requirements.txt^^^.
